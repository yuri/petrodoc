
Petrodoc is a tool for generating documenation from LuaRocks rockspecs and documentation from the same file.

See petrodoc file in this repository for an example of what a petrodoc spec would look like.

Usage:

    $ petrodoc.lua petrodoc

Output:

* doc/index.html (documentation)
* petrodoc-10.12.28.tar.gz (a release file)
* petrodoc-10.12.28-0.rockspec (a rockspec)

See http://spu.tnik.org/lib/petrodoc/ for more information.
